# Semiconductor Manufacturing ML Project

## 📊 Overview

This project focuses on predicting pass/fail outcomes in a semiconductor manufacturing process using machine learning techniques.

## 🔍 Key Steps

* Data cleaning and preprocessing
* Handling missing values
* Feature selection
* SMOTE for class balancing
* Model training (Logistic Regression, Random Forest, SVM)
* Hyperparameter tuning using GridSearchCV

## 🛠 Tools Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Seaborn
* Matplotlib

## 📈 Results

* Logistic Regression: ~93% accuracy
* Random Forest: ~99% accuracy
* SVM: ~99% accuracy

## 🧠 Key Learning

* Handling high-dimensional data
* Dealing with imbalanced datasets
* Model comparison and tuning
